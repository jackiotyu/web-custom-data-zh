# @vscode/web-custom-data 中文版

## 项目简介

本项目的目的是将 [@vscode/web-custom-data](https://github.com/microsoft/vscode-custom-data) 库的资源翻译成中文版，以方便中文用户使用。

## 项目结构

```
.
├── README.md               # 项目说明文件
├── data/          # 翻译后的中文数据文件
│   ├── html-data.json
│   └── css-data.json
├── src/                # 辅助脚本
└── package.json            # 项目配置文件
```
